\subsection{c\_capa3d}
\subsubsection{Relaciones}
\begin{itemize}
\item Es una especificación de \Clase{capa}.
\item Depende de \Clase{Objetos.objeto\_origen}.
\item Depende de \Clase{Objetos.camara}.
\end{itemize}
\subsubsection{Atributos}
\begin{itemize}
\item objeto\_origen : Objetos.objeto\_origen \newline
La raíz del árbol de objetos que se pintará en esta capa.
\item camara : Objetos.camara \newline
La cámara desde la que se verá la escena. La cámara debe pertenecer a la escena, es decir, debe estar en el mismo árbol de objetos del que objeto\_origen es la raíz.
\end{itemize}
\subsubsection{Métodos}
\begin{itemize}
\item +objeto\_origen : Objetos.objeto\_origen
\item +camara : Objetos.camara \newline
Getters.
\item +set\_objeto\_origen (ob : Objetos.objeto\_origen)
\item +set\_camara (cam : Objetos.camara)\newline
Setters. set\_camara comprueba que la cámara esté en la misma escena que el objeto origen.
\item +inicializador \newline
Comprueba que la cámara esté en la misma escena que el objeto origen.
\end{itemize}
\subsubsection{Parámetros de instanciación}
\begin{itemize}
\item ob:Objetos.objeto\_origen
\item cam:Objetos.camara \newline
Los atributos de la clase.
\end{itemize}

Las capas que componen la escena, en orden. Por defecto todas las capas son capas nulas.

\subsubsection{Métodos}
\begin{itemize}
\item capas : capa array
\item capa (i:int) : capa \newline
Getters. El número de capa varía entre 1 y num\_capas.
\item asignar\_capa (c:capa) (i:int) \newline
Setter. Establece la capa c en el puesto i de capas, donde i varía entre 1 y num\_capas.
\end{itemize}
\subsubsection{Parámetros de instanciación}
\begin{itemize}
\item num\_capas : int \newline
El número de capas que tiene esta escena.
\end{itemize}

\subsection{monitor}
\subsubsection{Relaciones}
\begin{itemize}
\item Es una especificación de \Clase{pantalla}.
\end{itemize}
\subsubsection{Atributos}
\begin{itemize}
\item dim\_x : int
\item dim\_y : int \newline
Las dimensiones de la pantalla en píxeles.
\item estado : t\_modo\_pantalla \newline
El estado de la pantalla, puede ser \emph{ventana} si la aplicación se está ejecutando como una ventana o \emph{pantalla\_completa} si la aplicación se está ejecutando sobre toda la pantalla.
\item estado\_ventana : t\_estado\_ventana \newline
El estado de la aplicación si se ejecuta en una ventana. Puede ser \emph{primer\_plano} si no hay ninguna aplicación por encima de ésta en el interfaz de ventanas, \emph{segundo\_plano} si hay alguna aplicación por encima o \emph{minimizada} si ha sido reducida a un icono en la barra de tareas.
\end{itemize}
\subsubsection{Métodos}
\begin{itemize}
\item dimensiones : (int * int) \newline
Devuelve el tamaño en píxeles de la pantalla.
\item \#set\_dimensiones (x:int) (y:int)
\item \#set\_estado (m:t\_modo\_pantalla)
\item \#set\_estado\_ventana (e:t\_estado\_ventana) \newline
Setters.
\item +estado : t\_modo\_pantalla
\item +estado\_ventana : t\_estado\_ventana \newline
Getters.
\item +cambiar\_modo (x:int) (y:int) (m:t\_modo\_pantalla)\newline
Cambia el modo gráfico de la pantalla, poniendo un modo con las dimensiones (x,y) y el tipo de ventana especificado.
\end{itemize}
